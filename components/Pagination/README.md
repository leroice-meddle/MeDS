# Pagination

Page number list with Previous / Next controls and gap indicator.

## Usage

```jsx
import Pagination from './Pagination'

<Pagination totalPages={68} initialPage={1} />
```

## Props

| Prop | Type | Default | Options |
|------|------|---------|---------|
| totalPages | number | 68 | — |
| initialPage | number | 1 | — |
