# Button Danger

Destructive action button. Two variants across two sizes.

## Usage

```jsx
import ButtonDanger from './ButtonDanger'

<ButtonDanger label="Delete" variant="Primary" size="Medium" />
<ButtonDanger label="Remove" variant="Subtle" size="Small" />
```

## Props

| Prop | Type | Default | Options |
|------|------|---------|---------|
| label | string | 'Button' | — |
| variant | string | 'Primary' | Primary, Subtle |
| size | string | 'Medium' | Medium, Small |
| disabled | boolean | false | — |
| onClick | function | — | — |
