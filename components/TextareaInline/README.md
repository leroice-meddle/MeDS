# Textarea Inline

Inline label + textarea for use inside forms.

## Usage

```jsx
import TextareaInline from './TextareaInline'

<TextareaInline label="Notes" value="Enter notes..." />
```

## Props

| Prop | Type | Default | Options |
|------|------|---------|---------|
| label | string | 'Label' | — |
| value | string | 'Value' | — |
| hasLabel | boolean | true | — |
| state | string | 'Default' | Default, Error, Disabled |
